# Free Fillable PDF Creator

A powerful, free web application that allows you to transform any PDF into a fillable form with interactive fields, signatures, and indicators.

## Features

### Form Fields
- **Text Fields**: Add fillable text input areas
- **Checkboxes**: Add interactive checkboxes
- **Signature Fields**: Add signature areas with support for drawn and typed signatures

### Customization Options
- Adjust field sizes with drag-and-drop and resize handles
- Customize colors (borders, backgrounds, text)
- Choose from multiple fonts (Helvetica, Times, Courier)
- Configure border widths
- Set fields as required

### Signature Management
- **Draw Signatures**: Use mouse or touch to draw signatures
- **Type Signatures**: Type your signature with multiple cursive font options
- **Save Signatures**: Store signatures in browser cache for reuse
- **Signature Library**: Access and manage your saved signatures

### Indicator Symbols
Add visual indicators to your PDFs:
- ✓ Check marks
- ✗ X marks
- → Arrows
- ★ Stars
- ! Important markers

### PDF Navigation
- Multi-page PDF support
- Page navigation controls
- Zoom in/out functionality (50% - 300%)
- Visual page indicators

## How to Use

1. **Upload PDF**: Click "Choose File" or drag-and-drop your PDF
2. **Add Fields**: Select a tool from the toolbar and draw on the PDF
3. **Customize**: Click any field to adjust properties in the right panel
4. **Create Signatures**: Click "Create Signature" to draw or type reusable signatures
5. **Download**: Click "Download Fillable PDF" to save your interactive form

## Technologies Used

- **PDF.js**: For rendering PDF documents
- **PDF-Lib**: For creating fillable PDF forms
- Pure HTML, CSS, and JavaScript (no frameworks required)
- LocalStorage for signature persistence

## Browser Support

Works in all modern browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

## Privacy

All processing happens locally in your browser. No files are uploaded to any server.
